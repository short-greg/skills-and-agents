# Agent Guidelines Intent

This directory contains setup guidelines for creating specialized subagents that can be delegated to by skills or directly by the AI assistant.

## Contents

- `code-review.md` - Code review agent setup guide
- `test-architect.md` - Test architect agent setup guide

## Relationship to Templates

Agent guidelines use [templates/agent-template.md](../templates/agent-template.md) as the base template for individual AGENT.md files created during setup.
